# 嘉南羊乳公關危機 - 詳細事實陳述

## 事件時間線與事實

### 第一階段：行銷公司自爆代操（2026年1月29日）

**事實陳述：**

1. 一家行銷公司在Threads上發文，自稱「代操嘉南羊乳Threads」[1]，並將該合作案例包裝為老品牌轉型的成功範例[2]

2. 該行銷公司的貼文曝光後引起網友譁然[2]，不少粉絲直呼「情感上難以接受」，質疑原本真實可愛的羊編形象是否只是行銷包裝[1]

3. 嘉南羊乳是台灣40年老品牌[3]，近期因Threads官方小編「羊編」勤奮海巡、自帶幽默感的互動風格在社群翻紅[4]

### 第二階段：羊編澄清與道歉（2026年1月29日晚間）

**事實陳述：**

1. 「羊編」在29日晚間親自發出長文澄清，強調「Threads上的每一篇發文、每一次海巡、每一則留言，都是由嘉南羊乳總公司本人執行」[5]

2. 羊編坦言行銷公司確實有參與，但僅提供圖像與視覺風格建議，並非帳號代操[5]

3. 羊編當晚一路海巡各大討論串，逐一誠懇回覆至凌晨2點，且每則道歉內容都不重複[6]

4. 羊編的澄清文短時間內成功扭轉風向，吸引超過3萬人按讚聲援[6]

5. 網友紛紛湧入支持，批評行銷公司的自誇行為是「最負面教材」，甚至發起訂閱運動力挺辛苦的小編[3]

### 第三階段：總公司發表鄭重聲明（2026年1月31日）

**事實陳述：**

1. 嘉南羊乳總公司在1月31日發布鄭重聲明，強硬力挺合作夥伴「開啟行銷」及「美樂蒂小姐」[7]

2. 聲明中明確指出，與「開啟品牌行銷顧問公司」擁有長期穩固的合約關係，視其為核心行銷顧問[8]

3. 聲明認證「羊IP」行銷規劃的創意發想與整體企劃，確實是由「開啟行銷」與「美樂蒂小姐」提供，本社審核後依程序推廣[8]

4. 聲明將網路上針對行銷團隊的指責定義為「不正當攻擊」與「惡意抹黑」，並直指背後有「有心人士」蓄意破壞商譽[8]

5. 聲明強調不排除與「開啟行銷」共同對相關人士提出妨害名譽之訴[8]

### 第四階段：網友反應與二度危機（2026年1月31日）

**事實陳述：**

1. 聲明發布後，原本力挺「羊編」的粉絲感到錯愕[8]

2. 網友紛紛留言表達失望：
   - 「竟然沒有要換行銷，完全沒看到行銷公司的作為，支持的朋友直接被當成傻子」[8]
   - 「這種聲明直接把羊編努力歸零了，我的天啊，這什麼公關災難」[8]
   - 「羊編是不是被換掉了」[8]
   - 「這個聲明看得出這間公司掌權的人大概是什麼年齡層」[8]
   - 「真心建議羊編離職，你的能力在別的地方會有更好的薪資」[8]
   - 「超敗好感，本來打算要訂羊奶了，現在看到這聲明覺得不用訂了」[8]
   - 「小編不眠不休海巡把形象拉回來，結果現在還要迎接第二波公關危機」[8]

3. 在Threads上，網友world.peace_chen發表了一篇獲得916個讚的貼文，表示對事件感到不舒服，決定不參與與避免接觸任何「嘉南羊乳」、「開啟行銷」與「美樂蒂小姐」今後任何行銷商品與行為[9]

4. 另一位網友在同一貼文下評論：「嘉南羊乳建議夏天再賣 再熱都被聲明弄成冰的」，獲得114個讚[9]

5. 網友emmachen5208評論：「我的想法和你一樣，覺得這麼會有這麼有事的總公司啊！這麼努力的要把員工努力做好的品牌好感，全部敗光」[9]

1. **嘉南羊乳（保證責任臺灣省嘉南羊乳運銷合作社）** - 台灣40年老牌羊乳品牌，總公司
2. **羊編** - 嘉南羊乳官方Threads小編，公司員工

## 核心爭議點

1. **真實性問題** - 行銷公司自爆代操，引發粉絲對品牌真實性的質疑
2. **信任危機** - 粉絲認為親切互動可能只是行銷包裝
3. **公司態度** - 總公司聲明力挺行銷公司，反而激怒支持者
4. **員工與公司的對立** - 形成「小編vs.總公司」的對立局面
5. **法律威脅** - 總公司揚言提告，引發網友進一步反感
