## Microsoft Foundry Local 發布日期確認

根據 Microsoft Azure 官方部落格 https://azure.microsoft.com/en-us/blog/microsoft-foundry-scale-innovation-on-a-modular-interoperable-and-secure-agent-stack/，該文章發布於 **2025 年 11 月 18 日**。

文章中提到：
- "Six months ago, we launched Foundry Local on Windows and Mac."（六個月前，我們在 Windows 和 Mac 上推出了 Foundry Local）
- "Foundry Local, now in private preview on Android, the world's most widely used mobile platform."（Foundry Local 現在在 Android 上進行私密預覽）

**重要發現**：
1. Foundry Local 在 Windows 和 Mac 上的首次發布是在 **2025 年 5 月左右**（文章發布於 11 月 18 日，往前推 6 個月）
2. Android 版本在 **2025 年 11 月 18-19 日**宣布進入私密預覽階段

**GitHub 倉庫資訊**：
- GitHub 倉庫：https://github.com/microsoft/foundry-local
- 最近的 commit 顯示活躍開發中
- 3 天前的 commit（2025-12-03 左右）
- 3 週前有多個更新

**結論**：
- Foundry Local 的 Windows/Mac 版本發布於 2025 年 5 月，不在時間範圍內
- Android 版本於 2025 年 11 月 18-19 日宣布私密預覽，也不在時間範圍內（2025-11-21 至 2025-12-06）
- 但 GitHub 倉庫在近期（11 月底至 12 月初）有持續更新

**評估**：
雖然主要發布不在時間範圍內，但考慮到：
1. 使用者特別提及此工具
2. GitHub 倉庫在時間範圍內有活躍更新
3. 這是 Microsoft 針對開發者的重要工具
4. Android 版本發布時間接近時間範圍邊界

**建議**：可以納入報告，但需要明確說明這是近期持續更新的工具，而非全新發布。
