## 11. React Server Components 存在嚴重遠端程式碼執行漏洞
- **來源**: React Blog
- **URL**: https://react.dev/blog/2025/12/03/critical-security-vulnerability-in-react-server-components
- **發布日期**: 2025 年 12 月 3 日
- **標題**: Critical Security Vulnerability in React Server Components
- **主要內容**:
  - React 官方披露了一個存在於 **React Server Components** 中的未經身份驗證的遠端程式碼執行漏洞，編號為 **CVE-2025-55182**，CVSS 評分高達 10.0。
  - 該漏洞源於 React 解碼發送到 **React Server Function** 端點的酬載 (payload) 時存在缺陷。即使應用程式未明確實現任何 Server Function，只要支援 Server Components 就可能受到影響。
  - 攻擊者可透過發送惡意 HTTP 請求，在伺服器端實現遠端程式碼執行。
  - **受影響的套件版本**：`react-server-dom-webpack`、`react-server-dom-parcel`、`react-server-dom-turbopack` 的 19.0.0、19.1.0、19.1.1 和 19.2.0 版本。
  - **已修復版本**：19.0.1、19.1.2 和 19.2.1。官方強烈建議立即升級。
  - 包含 Next.js、React Router、Waku 在內的多個框架和打包工具皆受影響，並已發布相應的更新指南。
- **權重評估**: 極高權重（CVSS 10.0、影響廣泛的前端生態系、需立即採取行動）

### FDA Qualifies First AI Drug Development Tool (AIM-NASH)
- **來源**: FDA (U.S. Food and Drug Administration)
- **URL**: https://www.fda.gov/drugs/drug-safety-and-availability/fda-qualifies-first-ai-drug-development-tool-will-be-used-mash-clinical-trials
- **發布日期**: 2025-12-08
- **重點**:
  - FDA 認證首個 AI 藥物開發工具 **AIM-NASH** (AI-Based Histologic Measurement of NASH)
  - 用於協助病理學家評估代謝功能障礙相關脂肪性肝炎 (MASH) 臨床試驗中的疾病活動
  - 這是一個基於雲端的工具，幫助病理學家對肝臟活檢組件進行評分
  - 包括脂肪浸潤（脂肪變性）、炎症（肝細胞氣球樣變和小葉炎症）和疤痕（纖維化）階段
  - MASH 是影響數百萬美國人的重大公共衛生挑戰
  - AIM-NASH 可幫助標準化組織學評估，並減少 MASH 藥物開發所需的時間和資源
  - AI 演算法分析肝臟活檢圖像，並根據 NASH 臨床研究網絡評分系統提供評分
  - 病理學家仍完全負責最終解釋，審查整個切片圖像和 AIM-NASH 輸出後接受或拒絕 AI 生成的評分
  - 基於全面驗證研究的資格認證，證明 AIM-NASH 輔助比較與專家共識的相似度與個別病理學家比較專家共識的相似度相當
