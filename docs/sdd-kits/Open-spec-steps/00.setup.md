# Step 00 - 核心概念與安裝初始化

> OpenSpec 三招：`proposal` → `apply` → `archive`
> 在寫程式前先定義規格，讓 AI 和你對「什麼叫完成」有共同認知。

## 安裝

```bash
npm install -g @fission-ai/openspec@latest
```

## 初始化

```bash
cd my-project
openspec init
```

選擇你使用的 AI 工具，系統會自動設定 Slash Command。

## 初始化後的目錄

```
openspec/
├── AGENTS.md     # 給 AI 讀的工作流程
├── project.md    # 專案基本資訊
├── changes/      # 變更提案
│   └── archive/  # 已完成的變更
└── specs/        # 目前系統的規格（真相來源）
```
