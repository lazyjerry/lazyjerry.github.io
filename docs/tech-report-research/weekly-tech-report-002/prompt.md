## 固定段落要求

1. 總結：100 至 300 字，依實際內容統整重點，需標記來源編號。
2. 亮點：依分類條列；每個條列先用一句話下結論，再用約 100 至 300 字說明；儘可能加入具體例子或譬喻；需標記來源編號。
3. 趨勢分析：依分類整理；依實際內容歸納趨勢或更深層觀察；儘可能加入具體例子或譬喻；需標記來源編號。
4. 參考資料：依分類與時間列出來源，欄位至少包含來源編號、文章標題、Google 搜尋快速連結、一句話繁體中文結論或亮點、文章發布日期、來源網址與來源網站名稱。

## 來源與權重規則

- 僅使用 2026-05-02 至 2026-05-16 內發布或可確認落在此期間的資料。
- 優先使用台灣主流媒體、通訊社、權威科技來源、官方公告與研究機構。
- 若來源無法確認是否在近 15 天內，直接排除。
- 若分類資料不足，必須補搜並在 `source.md` 保留候選來源與淘汰判斷。
- 來源編號格式：`[[{數字}. {資料來源名稱}]](#ref-{數字})`。
- 需遵守 `docs/tech-report-research/knowledge/source-priority.md`、`taiwan-media-landscape.md`、`date-citation-rules.md`、`report-format.md`。

## source.md 蒐集格式

每筆資料至少包含來源網址、繁體中文標題、100 至 500 字繁體中文摘要、權重分數（`1-10/10`）、發布日期、分類、是否納入正式參考資料。
