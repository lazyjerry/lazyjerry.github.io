## 四大段落要求

- 總結：100 至 300 字，依實際來源統整重點並標註來源編號。
- 亮點：依分類條列，每個條列先用一句話下結論，再用約 100 至 300 字說明，需標註來源編號。
- 趨勢分析：依分類整理，歸納可能趨勢或深層觀察，需標註來源編號。
- 參考資料：依分類與時間列出來源，欄位包含來源編號、文章標題、Google 搜尋快速連結、一句話繁體中文結論或亮點、發布日期、來源網址與來源網站名稱、分類。

## 來源與權重規則

- 僅使用 2026-05-22 至 2026-06-05 內發布或可確認落在此範圍內的資料。
- 優先使用有明確日期標記的來源；無法確認日期者排除。
- 優先採用台灣主流媒體、官方公告、權威科技來源與原始資料。
- 中國來源、社群平台與低可信度內容若無高權重來源可替代，直接排除。
- `source.md` 需保留候選來源與淘汰判斷。

## source.md 蒐集格式

每筆來源至少包含：

- 來源網址
- 繁體中文標題
- 100 至 500 字繁體中文摘要
- 權重分數（`1-10/10`）
- 發布日期
- 分類
- 是否納入正式參考資料
- 備註或淘汰原因

## 完成前檢查

- 各必要分類是否都有資料，若缺漏需明確註記原因。
- 資訊權重是否足夠，是否優先採用高權重來源。
- 搜尋時間範圍是否為 2026-05-22 至 2026-06-05。
- 資料來源是否正確，包含網址、來源網站名稱、發布日期與引用對應。
- 文章內引用的來源編號是否與參考資料表一致。
- 錯誤來源或整理內容是否已修正。
