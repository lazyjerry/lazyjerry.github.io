# 日期與引用規則

## 觸發時機

- 確認來源是否符合近 15 天規範時
- 建立文中引用編號時
- 製作超連結錨點與參考資料表時

## 內容

### 日期規則

- 來源請以有標記日期優先使用，並遵守今天往前 15 天內的時間範圍。
- 若來源未標記日期，且無法確定是否符合時間範圍，請不要使用。
- 若確認在 15 天內，但不確定精確發布時間，請在參考資料的「文章發布日期」欄位中明確寫上 `*未知時間`。
- 如未提及明確日期，以今天日期起算為主。

### 引用編號格式

- 文中引用請使用 `[[{數字}. {資料來源名稱}]](#ref-{數字})` 的格式。
- 參考資料區塊需使用表格形式呈現。
- 參考資料中的引用編號與內文需一致。

### 錨點規則

- 若 Markdown 超連結使用錨點，應在對應位置前加入 HTML 錨點。
- 格式範例：`<a id="122-分析框架"></a>`
