## 四大段落要求

1. 總結：100 至 300 字，依正式來源統整重點並標註來源編號。
2. 亮點：依分類條列，每則先下結論，再以 100 至 300 字說明，需標註來源編號。
3. 趨勢分析：依分類整理更深層觀察或趨勢，需標註來源編號。
4. 參考資料：以表格列出來源編號、文章標題與 Google 搜尋快速連結、一句話結論、發布日期、來源網址與來源網站名稱、分類。

## 來源與權重規則

- 僅納入 2026-06-20 至 2026-07-04 之間發布或更新且日期可確認的來源。
- 優先採用官方公告、權威科技媒體、台灣主流媒體與財經媒體。
- 中國來源、社群平台與日期不明來源需優先以高權重來源替換；無法替換時排除。
- 若分類資料不足，需補搜；若仍不足，於 notes.md 與 report.md 明確註記缺漏原因。

## source.md 蒐集格式

每筆候選來源至少包含：來源網址、繁體中文標題、100 至 500 字繁體中文摘要、權重分數（1-10/10）、發布日期、分類、是否納入正式參考資料，以及淘汰原因（如適用）。

## 完成前檢查

- 分類完整性：必要分類是否都有來源或缺漏說明。
- 資訊權重：正式參考資料是否以高權重來源為主。
- 搜尋時間範圍：每筆正式來源是否落在 2026-06-20 至 2026-07-04。
- 資料來源正確性：網址、來源網站名稱、發布日期與文章主張是否一致。
- 引用一致性：內文引用編號是否與參考資料表 `ref-*` 錨點一致。
- 錯誤修正狀態：已淘汰或修正的來源是否記錄於 source.md 或 notes.md。
