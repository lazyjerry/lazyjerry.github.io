## 總結

這期（2026-05-15～05-30）如果只抓主線，其實很明確：模型還在拚速度與入口，AI 基建繼續往電力、散熱與整櫃系統擴張，供應鏈資安風險則被 EVERY8D 事件狠狠放大。Google 在 I/O 2026 讓 Gemini 3.5 Flash 先於 Pro 上線，還直接放進 Google 搜尋，等於把輕量模型推到最大流量入口 [[1. 數位時代]](#ref-1)[[2. udn 科技玩家]](#ref-2)。另一頭，輝達單季營收來到 816 億美元，資料中心年增 92%，又用自研 Vera CPU 往更大的系統市場走，連財報分類都開始往邊緣運算重排 [[3. TechNews]](#ref-3)[[4. TechNews]](#ref-4)[[5. TechNews]](#ref-5)。基建端也很直接，液冷已經不是加分項，而是高功耗 AI 伺服器的基本配備；PwC 對電力、儲能與基礎建設的估算，也把這件事講得很白 [[6. 壹蘋新聞網]](#ref-6)[[13. TechNews]](#ref-13)。資安方面，EVERY8D 遭駭讓簡訊驗證、政府與金融供應鏈的脆弱點一次攤開 [[7. iThome]](#ref-7)[[8. iThome]](#ref-8)。另外，補查後也補齊了醫療科技與運輸物流的窗內來源，分別是台灣數位醫療 333 架構與無人機物流 24 條廊道 [[15. 中央社]](#ref-15)[[16. 中央社]](#ref-16)。

## 亮點

<a id="hl-ai"></a>
### AI 科技
Google 這次最值得注意的，不只是 Gemini 3.5 Flash 發表了，而是它先上場、還直接進了搜尋主流程。I/O 2026 上，Flash 先支撐 Gemini Spark、搜尋 AI Mode、Antigravity 與企業版 Gemini Enterprise；udn 也提到 Google 已把 Gemini 3.5 Flash 設成搜尋 AI 模式的預設基礎模型 [[1. 數位時代]](#ref-1)[[2. udn 科技玩家]](#ref-2)。這代表搜尋產品正在往「直接幫你做事」靠，而不是只丟一排連結給你。對靠搜尋流量吃飯的網站、內容方與工具服務，壓力會很實際。

<a id="hl-software"></a>
### 軟體工程
軟體工程這一欄的訊號也很集中。Gemini 3.5 Flash 在 Terminal-Bench 2.1 約 76.2%、在 MCP Atlas 約 83.6%，都高過前一代 Pro [[1. 數位時代]](#ref-1)。另一個更接地氣的例子是 LINE 臺灣團隊把 GitHub Spec Kit 與規格驅動開發（SDD）放進實際流程，讓 OA Plus 幫手市集的元件開發從數週縮到 3 天，資安與測試也更早進場 [[14. iThome]](#ref-14)。現在的模型已經不只是在 IDE 裡補幾行程式碼，而是開始接手規劃、呼叫工具、看錯誤再回修。工程師的重心也跟著往任務拆解、流程設計與結果審查移動。

<a id="hl-security"></a>
### 軟硬體資訊安全
EVERY8D 事件把一個老問題又拉回檯面上來：很多企業自認防守做得不差，但真正脆弱的地方可能在共用供應商。這家市占最高的企業簡訊平台疑似遭勒索軟體入侵，資料外洩、服務中斷超過一週，直接影響銀行、電商與金流業者的 OTP 與驗證流程 [[7. iThome]](#ref-7)。F-ISAC 隨後發布黃燈警訊，外洩樣本疑似牽涉政府機關與軍方簡訊日誌；同一時間，微軟也在修補 SharePoint 高風險漏洞 CVE-2026-45659 [[8. iThome]](#ref-8)。這類事件麻煩的地方就在這裡：不是單一公司出事，而是整條驗證鏈一起被拖下水。

<a id="hl-hardware"></a>
### 硬體或軟硬整合
輝達的方向現在已經很難誤讀了。它不想只賣 GPU，而是想把 CPU、GPU、DPU、網通一路包成整套系統。從財報來看，單季營收 816 億美元、資料中心年增 92%、網通業務年增 199%，成長動能還是很兇 [[3. TechNews]](#ref-3)。接著又用 Vera Rubin 平台把自研 Vera CPU 與 BlueField-4 推上檯面，直接碰 Intel 與 AMD 的地盤 [[4. TechNews]](#ref-4)。連財報分類都把「遊戲」併進「邊緣運算」，也看得出 GeForce GPU 的定位正在改寫，不再只是消費市場產品 [[5. TechNews]](#ref-5)。

<a id="hl-infrastructure"></a>
### 網路/伺服器等基礎
AI 伺服器的熱設計已經走到一個很務實的階段。當單櫃功耗逼近 200kW，氣冷壓不住就是壓不住，液冷自然從選配變成標配。嘉實多這類原本外界不會第一時間想到的公司，也開始推出冷卻液與熱管理方案，甚至往感測器與維運整合走 [[6. 壹蘋新聞網]](#ref-6)。這件事值得看，因為受惠的已經不只是散熱模組廠，冷卻液、感測器、服務維運都一起被帶起來，台灣供應鏈也卡在裡面。

<a id="hl-fintech"></a>
### 金融科技
金融科技這邊，討論焦點明顯不再只是交易平台本身。Bybit 執行長在 2026 高盛亞太金融科技大會提到，資產代幣化的推進速度可能比市場原先想得更快，驅動力來自穩定幣、RWA 與機構採用 [[9. PChome 新聞]](#ref-9)。後面真正有商業價值的，會是支付、託管、抵押品管理、收益產品與合規基礎設施。換句話說，市場開始在談「怎麼把這套東西變成金融機構能真正用的系統」。

<a id="hl-healthtech"></a>
### 醫療科技
醫療科技這期補進來的重點，不在單一醫院導入了什麼新工具，而是台灣想把數位醫療拉到更上層的公共架構。中央社報導，衛福部提出的「333 架構」涵蓋三大空間、三大標準與三大 AI 中心；家醫大平台已導入 AI 風險預測，MediCloud 也串起即時就醫、用藥資訊與影像判讀 [[15. 中央社]](#ref-15)。如果這條線能走下去，智慧醫療談的就不只是點狀導入，而是資料互通與臨床流程能不能接起來。

<a id="hl-logistics"></a>
### 運輸物流
無人機物流在台灣也不是停在展示階段了。民航局 5 月 26 日舉辦相關輔導會議時提到，台灣已累積 24 條無人機物流廊道，接下來要補的是法規與作業制度 [[16. 中央社]](#ref-16)。這個進度比單次試飛更有意思，因為物流能不能做大，最後還是看航線、空域、責任歸屬與營運規則能不能被穩定複製。

<a id="hl-proptech"></a>
### 房地產與室內外裝潢
AI 帶來的外溢效應，連產業用地都看得到了。新北國際 AI+智慧園區在 5 月 15 日公告標售 16 筆第三種產業專用區土地，總面積約 9,081 坪 [[11. 中時房市]](#ref-11)。報導特別強調交通條件、周邊工業區與科技產業聚落，顯示地方政府已經把 AI 需求直接包進土地招商敘事。這不只是晶片與伺服器的故事，也會往園區開發、廠辦規劃與地方投資策略延伸。

<a id="hl-arts"></a>
### 現場表演藝術
表演藝術這則我認為很值得留。太魯閣族儀式傳承者東冬・侯溫與鄭淑麗合作《遊林驚夢：巧遇 Hagay》，在北藝中心球劇場把數位影像、雷射、動作捕捉、即時編碼、神話歌謠與原民樂舞放在同一個表演裡 [[12. PAR 表演藝術雜誌]](#ref-12)。重點不只是技術堆疊，而是這些工具真的被拿來處理文化記憶與祭儀敘事。這類作品會讓人比較願意認真看「科技怎麼服務內容」，而不是只看它有多炫。

<a id="hl-entertainment"></a>
### 影視音樂
Spotify 這波更新，真正有意思的是它開始把生成式 AI 與二次創作放進正式的商業框架。2026 投資人日提到的功能包含 Reserved、合法翻唱與重混分潤、Personal Podcasts、Prompted Playlists、Studio by Spotify Labs，以及 Memberships 創作者訂閱 [[10. udn 女子漾]](#ref-10)。功能多不多是一回事，更關鍵的是平台願意先把授權與分潤規則講清楚。AI 時代的音樂版權，大概就是往這個方向走。

<a id="hl-others"></a>
### 其他領域科技應用突破
如果要把這期很多新聞連起來看，最後還是會回到電力。TechNews 引述 PwC 報告指出，到 2050 年全球基礎建設投資可能達 151.1 兆美元，AI 帶來的高用電需求正把電力、儲能與電網升級推到前面，再生能源、核能與儲能投資都在往上走，其中儲能成長上看 273% [[13. TechNews]](#ref-13)。模型、晶片、資料中心講到最後，都還是要看誰拿得到穩定電力，誰撐得住用電成本。

<a id="trends"></a>
## 趨勢分析

<a id="tr-ai"></a>
### AI 科技
模型競爭現在越來越不像單純比智商大賽，比的是誰跑得快、誰工具接得順、誰又能撐住成本。Google 讓 Flash 先於 Pro 出場，還直接放進搜尋，就是把低延遲、高吞吐的模型推到最大入口 [[1. 數位時代]](#ref-1)[[2. udn 科技玩家]](#ref-2)。再對照輝達資料中心與網通業務的爆發 [[3. TechNews]](#ref-3)，很容易看出一條分工線：前端模型要更輕、更快；後端基建則繼續往大規模投資走。

<a id="tr-software"></a>
### 軟體工程
代理化幾乎已經回不去了。當模型發表會開始把 Terminal-Bench、MCP Atlas 這種偏實作能力的評測搬上檯面 [[1. 數位時代]](#ref-1)，加上 LINE 臺灣已把 GitHub Spec Kit 與 SDD 放進產品流程 [[14. iThome]](#ref-14)，工程團隊的生產力焦點就不再只是寫得快，而是拆得準、接得順、審得住。接下來 CI/CD、測試與錯誤修復很可能會有更多環節交給代理處理；工程師自己要守的，會是目標、限制與品質邊界。搜尋也越來越代理化後 [[2. udn 科技玩家]](#ref-2)，文件與 API 能不能讓機器順利讀懂，也會變成現實要求。

<a id="tr-security"></a>
### 軟硬體資訊安全
供應鏈與第三方服務，大概會是 2026 年很多資安事件的共同破口。EVERY8D 事件已經證明，攻擊者不一定要一間一間打銀行，只要先拿下共用的簡訊或 OTP 供應商，就能讓整條身分驗證鏈一起卡住 [[7. iThome]](#ref-7)[[8. iThome]](#ref-8)。接下來企業勢必要重新盤點外部關鍵相依，要求供應商交出更具體的韌性證明，也得替 OTP 這類單點服務準備備援。SharePoint 漏洞同步被修補 [[8. iThome]](#ref-8)，也提醒大家別把內部協作工具排除在攻擊面之外。

<a id="tr-hardware"></a>
### 硬體或軟硬整合
輝達現在賣的越來越像一整座機櫃，而不是單一晶片。CPU、GPU、DPU 到網通一路垂直整合，讓資料中心用系統而不是零件下單 [[3. TechNews]](#ref-3)[[4. TechNews]](#ref-4)。再把遊戲業務併進邊緣運算後，消費級 GPU 也被重新定位為終端 AI 與機器人的運算底座 [[5. TechNews]](#ref-5)。對台灣供應鏈來說，機會自然會往系統組裝、互連與散熱整合移動，誰能交整櫃方案，誰就更接近核心價值。

<a id="tr-infrastructure"></a>
### 網路/伺服器等基礎
液冷已經不只是散熱方案，而是整個資料中心規劃的一部分。嘉實多從冷卻液、感測器切進來，也代表競爭重心正在從硬體模組往耗材與服務擴散 [[6. 壹蘋新聞網]](#ref-6)。再配上 PwC 對電力與儲能投資的估算 [[13. TechNews]](#ref-13)，可以預期資料中心的選址、管路、供電與 PUE 設計都會更早被拉進決策流程。之後比的不是只有算力，還有你能不能穩定供電、把熱處理好。

<a id="tr-fintech"></a>
### 金融科技
代幣化要真的走進主流金融，最後一定得接上機構流程。當穩定幣被拿來做結算層、RWA 把傳統資產搬上鏈，重點自然從散戶炒作移到支付、託管、合規、抵押品管理與清結算 [[9. PChome 新聞]](#ref-9)。對開發者來說，鏈上身分、KYC／AML 模組、資產託管與跨鏈結算 API 這些比較「無聊但必要」的東西，反而更可能先變成需求。

<a id="tr-healthtech"></a>
### 醫療科技
台灣智慧醫療接下來的問題，比較不像「技術做不做得出來」，而是制度能不能跟上。333 架構把資料空間、交換標準與 AI 中心放在同一張藍圖上，家醫大平台與 MediCloud 也已經把風險預測、用藥資訊與影像判讀接進臨床場景 [[15. 中央社]](#ref-15)。後面真正卡關的，多半還是健保給付、TFDA 取證、本土臨床驗證與 FHIR 資料互通能不能一起往前走。少一段，整體就會卡住。

<a id="tr-logistics"></a>
### 運輸物流
運輸科技的重點，正在從「飛不飛得起來」轉成「能不能持續營運」。24 條無人機物流廊道，代表主管機關已經開始把示範案整理成比較可複製的航線與作業規則 [[16. 中央社]](#ref-16)。接下來真正的瓶頸還是在空域管理、保險責任、偏鄉與離島的服務模式，以及物流業者願不願意把無人機接進原本系統。這時候航線調度、遠端監控、任務追蹤與異常通報，才會變成實際需求。

<a id="tr-proptech"></a>
### 房地產與室內外裝潢
AI 也在重寫產業用地的估值邏輯。新北國際 AI+智慧園區標售第三種產業專用區土地時，特別把交通條件、工業區與 ASML 等產業聚落放進賣點 [[11. 中時房市]](#ref-11)。這表示地方招商已經不只是賣地，而是賣一整套和 AI 產業相容的基礎條件。對營建與裝修市場來說，智慧建築、能源監控、儲能與 IoT 整合，之後只會越來越常出現在園區與廠辦規劃裡。

<a id="tr-arts"></a>
### 現場表演藝術
表演科技成熟之後，沉浸式作品也開始比較少停在視覺奇觀。像這次的案例，雷射、數位影像、動作捕捉與即時編碼被拿來承載文化記憶與在地敘事 [[12. PAR 表演藝術雜誌]](#ref-12)，不是單純拿來炫技。之後劇場與科技團隊的協作應該會更常見，VR、MR 與即時動作捕捉也會更自然地進到製作流程，舞台技術職能跟著變動幾乎是必然。

<a id="tr-entertainment"></a>
### 影視音樂
內容平台現在搶的，很大一塊就是誰先把生成式 AI 與二次創作的規則訂清楚。Spotify 用授權與分潤把二創拉進正規市場 [[10. udn 女子漾]](#ref-10)，本質上是在用商業設計處理版權爭議。接下來其他平台大概也會陸續推出合法 AI 創作與收益拆分機制，而著作權法怎麼看待「人類創意介入」的程度，會直接影響收益最後怎麼分。

<a id="tr-others"></a>
### 其他領域科技應用突破
